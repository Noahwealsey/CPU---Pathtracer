# CPU---Pathtracer
A physically-based path tracer in C++ featuring Monte Carlo integration, cosine-weighted sampling, and recursive global illumination.
Built from scratch to understand light transport and rendering fundamentals.

How to Run:

after compiling the code and running the .exe you will need to convert the resulting .ppm with
a .png converter you should use Magick for that and done you will get a nice 3d render
